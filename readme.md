# Why Tool Learning?
人类拥有创造工具和使用工具的能力，从而突破物理空间的约束。大语言模型具备指令跟随、推理、规划等方面的能力，具有像人类一样使用工具的潜力。基于此，工具学习旨在结合工具和语言模型来提高准确性、效率以及自动化解决问题的能力。
